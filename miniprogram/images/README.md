## 图标规格建议
- 尺寸：建议使用 64x64 像素
- 格式：PNG格式，支持透明背景
- 风格：简洁现代，与整体设计风格一致
- 颜色：主色调使用绿色 (#3cc51f)

## 临时解决方案
如果暂时没有这些图标，可以：
1. 使用在线图标库（如 iconfont）下载对应图标
2. 使用占位图片进行测试
3. 在代码中暂时注释掉图标引用，使用文字替代
